Stack Tecnologico

Una panoramica dei linguaggi, framework e strumenti che utilizzo per dare vita ai progetti.

Linguaggi di Programmazione

Le fondamenta solide su cui costruisco logica ed ecosistemi digitali.

go

Linguaggio di programmazione open-source sviluppato da Google, celebre per la sua efficienza, gestione della concorrenza e velocità di esecuzione.

html

Linguaggio di markup fondamentale per strutturare e formattare i contenuti delle pagine web.

javascript

Linguaggio di scripting lato client e server indispensabile per rendere i siti web interattivi e dinamici.

php

Linguaggio di programmazione lato server ampiamente utilizzato per lo sviluppo di siti web dinamici e applicazioni web.

css

Linguaggio utilizzato per definire la formattazione, il design e il layout visivo dei documenti HTML.

sass

Estensione e preprocessore CSS che aggiunge funzionalità avanzate come variabili, cicli e nidificazione alla scrittura degli stili.

Framework

Strumenti avanzati per lo sviluppo di interfacce veloci e reattive.

bootstrap

Framework CSS e JavaScript tra i più famosi al mondo per la creazione di interfacce web responsive e mobile-first.

flutter

Framework di Google per lo sviluppo di applicazioni compilate nativamente per mobile, web e desktop da un unico codice sorgente.

nextjs

Framework React avanzato che permette il rendering lato server (SSR) e la generazione di siti statici ottimizzati.

react

Libreria JavaScript per la creazione di interfacce utente reattive e basate su componenti riutilizzabili.

svelte

Framework JavaScript innovativo che sposta il lavoro di compilazione al build-time, producendo codice nativo ultraleggero.

symfony

Framework PHP robusto e strutturato ad alte prestazioni, ideale per lo sviluppo di applicazioni aziendali complesse.

tailwind

Framework CSS basato su classi di utilità che permette di progettare interfacce grafiche personalizzate direttamente nel codice HTML.

Backend & Database

Gestione dei dati, API sicure e architetture server performanti.

gnu-bash

Interprete di comandi e linguaggio di scripting standard per sistemi operativi Unix e Linux.

jetbrains

Ambiente di sviluppo software e suite di IDE professionali per programmatori.

linux

Sistema operativo open-source che costituisce l'infrastruttura fondamentale per server e ambienti di sviluppo.

mongodb

Database NoSQL orientato ai documenti che memorizza i dati in formato flessibile simile a JSON.

mysql

Uno dei sistemi di gestione di database relazionali (RDBMS) open-source più diffusi al mondo.

phpmyadmin

Strumento di amministrazione web grafico per la gestione semplificata di database MySQL e MariaDB.

plesk

Piattaforma di Web Hosting e pannello di controllo per la gestione semplificata di server e siti web.

postgresql

Potente sistema di database relazionale open-source ad oggetti, noto per l'affidabilità e l'estendibilità.

sql-lite

Libreria software che fornisce un sistema di gestione di database relazionali leggero, autonomo e senza server.

wordpress

Il sistema di gestione dei contenuti (CMS) più popolare al mondo per la creazione e gestione di siti web e blog.
